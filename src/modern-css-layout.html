现代CSS布局技巧

Flexbox和Grid彻底改变了网页布局。本文将介绍这两种布局技术的核心概念和实用技巧。

Flexbox布局

Flexbox（弹性盒子）是CSS3引入的一维布局模型，非常适合在单一方向上布局元素。

基本概念

.container {
    display: flex;
    flex-direction: row; /* 主轴方向 */
    justify-content: center; /* 主轴对齐 */
    align-items: center; /* 交叉轴对齐 */
    gap: 16px; /* 间距 */
}

.item {
    flex: 1; /* 弹性放大 */
    flex-shrink: 0; /* 不收缩 */
}

实用技巧

  • 居中对齐：`justify-content: center; align-items: center;`
  • 两端对齐：`justify-content: space-between;`
  • 自动填充：`flex-wrap: wrap;`
  • 垂直居中：`align-items: center;`

Grid布局

Grid（网格布局）是CSS3引入的二维布局模型，可以同时在行和列方向上布局元素。

基本概念

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3列等宽 */
    grid-template-rows: auto; /* 行高自动 */
    gap: 20px; /* 间距 */
    grid-template-areas: 
        "header header header"
        "sidebar content content"
        "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

实用技巧

  • 自适应列数：`grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));`
  • 命名区域：使用`grid-template-areas`创建语义化布局
  • 自动填充：`grid-auto-flow: dense;`
  • 对齐控制：`place-items: center;`

布局组合技巧

1. 圣杯布局

.layout {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    grid-template-areas: 
        "header header header"
        "nav main aside"
        "footer footer footer";
    min-height: 100vh;
}

2. 响应式卡片网格

.card-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 24px;
}

@media (max-width: 768px) {
    .card-grid {
        grid-template-columns: 1fr;
    }
}

现代布局最佳实践

  • 优先使用Flexbox处理一维布局
  • 使用Grid处理二维布局
  • 结合使用实现复杂布局
  • 使用CSS变量提高可维护性
  • 使用容器查询实现组件级响应式

结语

Flexbox和Grid彻底改变了CSS布局的方式，让我们能够用更少的代码实现更复杂的布局效果。掌握这两种技术，是现代前端开发者的必备技能。
